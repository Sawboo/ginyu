Ginyu
=====

A simple but robust blogging application for Django.

Features
----

- [x] tagging support
- [ ] comments via Disqus
- [ ] integrated search
- [x] use of Markdown as markup format
- [x] archive views
- [ ] RSS feed support
- [x] post drafts and prepublication
- [ ] bit.ly support
- [x] admin dashboard
- [ ] twitter integration
